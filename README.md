# 📊 Análisis de Desempeño de Tiendas - Proyecto de Ciencia de Datos

## 📌 Descripción del Proyecto
Este proyecto tiene como objetivo analizar el desempeño de cuatro tiendas pertenecientes al Sr. Juan con el fin de determinar cuál presenta el menor rendimiento y debería considerarse para su venta.

Para ello se realizó un análisis exploratorio de datos utilizando Python en Google Colab, evaluando diferentes indicadores de desempeño como ingresos, categorías de productos, satisfacción de clientes y costos de envío.

El análisis permite tomar una decisión basada en datos para optimizar la gestión del negocio.

---

## 🎯 Objetivo
Identificar cuál de las cuatro tiendas presenta el peor desempeño general y recomendar su venta, considerando diferentes métricas comerciales.

---

## 📊 Factores Analizados

Durante el análisis se evaluaron los siguientes aspectos:

- 💰 Ingresos totales por tienda
- 📦 Categorías de productos más y menos vendidas
- ⭐ Calificaciones promedio de los clientes
- 🛒 Productos más y menos vendidos
- 🚚 Costo promedio de envío por tienda

Estos indicadores permiten comparar el rendimiento de cada tienda desde diferentes perspectivas.

---

## 🛠 Tecnologías Utilizadas

- Python
- Google Colab
- Pandas
- Matplotlib
- Seaborn

Estas herramientas se utilizaron para el procesamiento de datos, análisis estadístico y visualización de resultados.

---

## 📈 Análisis Realizado

El proyecto incluyó las siguientes etapas:

1. Importación y exploración de datos
2. Limpieza y preparación del dataset
3. Análisis de ingresos por tienda
4. Análisis de ventas por categoría
5. Evaluación de calificaciones de clientes
6. Identificación de productos más y menos vendidos
7. Análisis de costos de envío
8. Visualización de resultados mediante gráficos

---

## 🧾 Conclusión

Después de analizar todos los indicadores, se identificó que la **Tienda 4 presenta el desempeño más bajo en comparación con las demás tiendas**.

Los resultados muestran que esta tienda presenta menores ingresos y un rendimiento general inferior en varios de los factores analizados.

Por esta razón, se recomienda que el **Sr. Juan considere vender la Tienda 4**, permitiendo enfocar recursos y esfuerzos en las tiendas con mejor desempeño.
